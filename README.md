# Fraud Detection Dashboard using SQL & Python

## Overview

This project demonstrates how SQL and Python can be used to analyze transaction data and identify suspicious patterns.

## Tools Used

- SQL
- Python
- Pandas
- Microsoft Excel
- Google Colab

## Dataset

- 100,000 simulated transaction records
- Generated using Python

## Project Features

- Analyzed transaction data using SQL
- Detected high-value transactions
- Identified customers with frequent transactions
- Summarized transaction amount by city
- Identified suspicious IP addresses
- Created Excel Pivot Table and Chart
- Performed data analysis using Python (Pandas)
